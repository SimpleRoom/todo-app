### 规约开发步骤

```

/speckit.constitution 定底线边界：这是一个任务管理系统，中文界面，清新简洁，纯前端localStorage持久化
生成项目宪章文档

/speckit.specify 写需求：实现一个任务清单引用，功能包括：新增任务(标题和描述)、删除任务、标记完成/取消完成、状态筛选(全部、
  未完成、已完成)、localStorage持久化
  
/speckit.clarify 补充盲区：进行需求澄清

/speckit.plan  出方案：

/speckit.tasks 拆任务：

/speckit.analyze 查一致：扫描需求、方案、任务直接是否有遗漏

/speckit.implement 出代码：
```
